# 测试输入：AI 中间层大洗牌

请使用 `outline-to-writing-cards` Skill 处理以下长篇文章计划。

## 选题

AI 生态已经进入洗牌阶段。最聪明的模型和最便宜的模型已经占据两端，真正危险的是广泛的中间层。

## 当前核心判断

一家模型公司是不是中间层，不由它自己定义，而由资本成本、模型能力、价格和用户付费共同决定。

## 当前大纲与材料

1. AI 行业为什么出现 K 型分化；
2. 前沿大模型公司的资金瀑布：用户每付 1 美元，公司如何花；
3. 模型价格下降时间线；
4. 模型能力与价格的对照；
5. 中国大模型价格战；
6. 资本为什么只愿意投极少数前沿公司；
7. 中间层公司的伤亡时间线；
8. 01.AI 从基础模型竞争转向“小而美”；
9. AI21 Labs 裁员并转向 agent 优化；
10. Inflection、Adept、Character.AI 被大厂收编；
11. 百川转向医疗；
12. Aleph Alpha、Cohere、Mistral 转向主权 AI；
13. 智谱、MiniMax 上市；
14. 月之暗面继续融资；
15. 上岸不等于盈利，只是拿到继续烧钱的资格；
16. B 端和 C 端需求分化；
17. 数据、工作流、分发、任务路由可能成为第三条路；
18. 中间层未来的四种结局。

## 已有兴趣判断

我真正感兴趣的不是列举哪些公司失败，而是：

- 为什么很多公司看起来还活着，实际上已经退出了原来的战争；
- 为什么“转型”有时是承认自己已经失去原赛道的位置；
- 为什么“上市、融资、政府订单”只能证明它获得了续命权，不能证明商业模式成立。
